Introduction

Dynaphopy allows to calculate anharmonic phonon linewidhts and frequency shifts using the mode descomposition technique. To do that it uses a VASP calculated Molecular Dynamics (MD) output and projects the atoms velocity into a set of harmonic phonon modes obtained by Phonopy. Afterwards the power spectrum is calculated usign Fourier transform or Maximum Entropy Method (MEM) and the peaks are fitted to Lorentzian function to extract the phonon properties. $$ \left\langle {{V^*}\left(0 \right)V\left( t \right)} \right\rangle = \int\limits_{t' = 0}^\infty {{V^*} \left( {t'} \right) \cdot V\left( {t' + t} \right)dt'} $$
